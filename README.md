# Full Stack Chat App 

## 🧠 Tech Stack

* React (Vite)
* FastAPI
* Postman (API testing)

## ✨ Features

* Frontend + Backend integration
* REST API with FastAPI
* GET & POST endpoints
* Chat interaction system

## ⚙️ How to Run

### Frontend

cd frontend
npm install
npm run dev

### Backend

cd backend
python -m venv venv
venv\Scripts\activate
pip install fastapi uvicorn
uvicorn main:app --reload

## 📌 API Endpoints

* GET `/` → Check backend
* POST `/chat` → Send message
